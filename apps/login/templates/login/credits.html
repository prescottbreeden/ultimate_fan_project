About Ultimate Fan

Ultimate Fan is a sports trivia game that allows users to test their knowledge on famous athletes.

All trivia questions were generated via web scraping with the Python Beautiful Soup module, which searched various websites for data on athletes. We then wrote an algorithm to convert this data into trivia questions that were then transcribed into a csv file before being uploaded into our database. Registration and login utilize full password hashing with salt via the bcrypt module and REGEX for email identification.

Select the ‘Quizzes’ tab to test your knowledge!!

Technologies utilized for this project include:

Python3.6, Django 2.0.3, SQLite3, jQuery3.3.1, CSS3 (all animations handled by Keyframes), ChartJS
